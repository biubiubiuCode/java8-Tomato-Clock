# Tomato Clock 番茄时钟
## 介绍
番茄APP为了解决长时间使用电脑而忘记休息、护眼、喝水的问题而设计，番茄APP运行在Chrome上，
所以不管你是用window、Mac还是linux，只要装有Chrome浏览器的地方就能运行番茄APP。页面
经过我精心设计，我大体满意了。希望番茄APP可以帮你提高工作学习效率的同时带给你健康办公的习惯。
